# Battery Cells Card v1.0.0

Initial release

## Features
- Display battery cell voltages in 2-column grid
- Color-code min (red) and max (green) voltages  
- Show voltage delta in mV
- Support 1-32 cells per battery
- Efficient single calculation of min/max values
- 97% reduction in template calculations

## Installation
Install via HACS or manually copy to www/ folder
